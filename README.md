# Predictive Machine Maintenance using Machine Learning

This project predicts machine maintenance requirements using machine learning models.
It helps industries reduce downtime and improve operational efficiency.

## Features
- Machine failure prediction
- ML model training & evaluation
- Flask-based web interface
- Real-time input prediction

## Tech Stack
- Python
- Machine Learning
- Flask
- Scikit-learn
- Pandas, NumPy
- HTML, CSS

## How to Run
```bash
pip install -r requirements.txt
python app.py
